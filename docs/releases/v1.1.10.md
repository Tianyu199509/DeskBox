# DeskBox 1.1.10

## English

DeskBox 1.1.10 is a small stability release before the next architecture refactor. It fixes Quick Capture recent clipboard monitoring after restart and restores scrolling when the recent list becomes longer than the widget height.

### Download

- [DeskBox_Setup_1.1.10_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.1.10/DeskBox_Setup_1.1.10_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Fixed Quick Capture recent clipboard monitoring after restart by initializing clipboard event listening on the UI thread.
- Fixed Quick Capture recent list scrolling when the list is longer than the widget.
- Added the phase-1 widget architecture refactoring plan as the baseline for the next development cycle.

### Compatibility

- Target system: Windows 11. Windows 10 is not the primary target, but this release keeps the existing drag/drop diagnostics and repair flow.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

## 中文

DeskBox 1.1.10 是开始下一轮架构重构前的小型稳定版本，主要修复随记最近复制内容重启后偶发不自动记录，以及最近列表内容过多时无法滚动的问题。

### 下载

- [DeskBox_Setup_1.1.10_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.1.10/DeskBox_Setup_1.1.10_x64.exe)

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

### 主要更新

- 修复随记最近复制内容在重启后偶发不自动记录的问题，剪贴板事件监听现在会在 UI 线程初始化。
- 修复随记最近内容过多时列表无法上下滚动的问题。
- 新增第一阶段格子架构重构路线文档，作为下一轮开发前的稳定基线。

### 兼容性

- 目标系统：Windows 11。Windows 10 仍不是主要目标，但保留现有拖拽诊断和一键修复流程。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、格子配置、随记数据和收纳目录内容。
