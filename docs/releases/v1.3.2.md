# DeskBox 1.3.2

DeskBox 1.3.2 is a major feature release. It introduces a complete desktop search system (USN Journal + Windows Index), adds three new languages (Japanese, German, Brazilian Portuguese) with a rewritten localization architecture, redesigns the onboarding flow, adds a hardware-adaptive tray animation system, expands the weather city database 10×, ships an ARM64 installer, and fixes a critical QuickLook compatibility issue.

## Highlights

### Search System (New)

- Full-text desktop search with a dedicated popup window, global hotkey activation, and instant results.
- USN Journal indexing for fast file discovery without directory-tree walking.
- Windows Index integration for content-searchable results.
- Weighted result ranking, search history, type sort/filter, and a recommended-apps panel with keyboard navigation.
- Dedicated search settings section.

### Multi-Language Expansion

- Added Japanese (ja-JP), German (de-DE), and Brazilian Portuguese (pt-BR) with 1500+ strings each.
- Replaced legacy `.resw` with a JSON-based `LocalizationService` for easier translation contributions.

### Onboarding Redesign

- Rebuilt as a focused five-step flow (Welcome → Features → Storage → Hotkey → Done) with entrance animations and full five-language support.

### Adaptive Tray Animation System

- Hardware-adaptive animation controller that adjusts complexity based on GPU capability.
- Batch animation driver for synchronized multi-widget show/hide.
- Synchronized scheduling to eliminate tearing on rapid toggles.

### Weather Improvements

- City database expanded from ~500 to 5000+ entries with CJK-aware fuzzy search.
- Improved location service with timeout handling and last-known-position fallback.
- Refined widget layouts for all four size modes.

### Installer and Updater

- ARM64 installer support.
- Reliable process-kill before overwrite, force-update enforcement, helper cleanup, official download URLs.
- English installer language file for non-Chinese systems.

### Architecture

- Extracted `App.Tray.cs`, added `ServiceRegistry`, `SettingsMigrationService`, `FileMetaService`, `AppDiagnosticsService`.
- Added a 10-chapter user guide under `docs/user-guide/`.

### Fixes

- **QuickLook compatibility (critical)**: Fixed pipe server crash caused by empty connections. Availability checks no longer touch the named pipe.
- **Wallpaper loss**: Prevented `WorkerW` spawn from destroying the desktop wallpaper.
- **Capsule defaults sync**, **tray menu height**, **dead shortcut cleanup**, **weather code mapping**.

### Installer, Search & Capsule Polish

- **Installer language selection**: The installer now shows a language-selection dialog (Chinese, English, Japanese, German, Brazilian Portuguese) pre-selected to the system locale. The chosen language is written to `HKCU\Software\DeskBox\InstallLanguage`, and DeskBox uses it as the default app language on first run (a manual in-app change still wins).
- **Search popup polish**: The result-list header now aligns with the data rows; the sort header carries a subtle background and shares the menu-bar margins.
- **Weather capsule fix**: Removed a duplicate title icon so capsule mode shows only the weather emoji.
- **Capsule hover mask**: Hidden the semi-transparent right-edge hover mask in capsule mode (interaction unchanged).

## 中文说明

DeskBox 1.3.2 是一个重大功能版本。引入完整的桌面搜索系统（USN 日志 + Windows 索引），新增三种语言（日语、德语、巴西葡萄牙语）并重写本地化架构，重构引导流程，新增硬件自适应托盘动画系统，天气城市数据库扩展 10 倍，发布 ARM64 安装器，并修复 QuickLook 兼容性严重问题。

### 更新重点

#### 搜索系统（全新）

- 全文桌面搜索：专用搜索弹窗、全局快捷键唤起、即时结果展示。
- USN 日志索引 + Windows 索引集成，快速文件发现。
- 加权结果排序、搜索历史、类型排序/筛选、推荐应用面板（键盘导航）。
- 独立搜索设置分区。

#### 多语言扩展

- 新增日语、德语、巴西葡萄牙语，每种 1500+ 条目。
- 本地化架构从 `.resw` 重写为 JSON 嵌入式资源。

#### 新用户引导重构

- 五步聚焦流程（欢迎 → 功能 → 收纳 → 快捷键 → 完成），入场动画 + 五语言支持。

#### 自适应托盘动画系统

- 硬件自适应控制器，根据 GPU 能力调整动画复杂度。
- 批量动画驱动器，同步多格子显示/隐藏。
- 同步调度消除快速切换撕裂。

#### 天气改进

- 城市数据库 500 → 5000+，CJK 模糊搜索。
- 定位服务优化 + 四种尺寸布局改进。

#### 安装器与更新器

- ARM64 安装器、可靠进程关闭、强制更新、缓存清理、正式下载地址、英文安装界面。

#### 架构

- 提取 `App.Tray.cs`，新增 `ServiceRegistry`、`SettingsMigrationService`、`FileMetaService`、`AppDiagnosticsService`。
- 新增 10 章用户指南。

#### 修复

- **QuickLook 兼容性（严重）**：修复空连接导致管道服务器崩溃，可用性检查不再触碰管道。
- **壁纸丢失**、**胶囊默认值对齐**、**托盘菜单高度**、**死快捷方式清理**、**天气代码映射**。

#### 安装器、搜索与胶囊打磨

- **安装器语言选择**：安装器现在提供语言选择对话框（中文、英文、日语、德语、巴西葡萄牙语），默认按系统区域预选。所选语言写入 `HKCU\Software\DeskBox\InstallLanguage`，DeskBox 首次启动会默认使用该语言（手动在应用内切换仍优先）。
- **搜索弹窗打磨**：结果列表表头与数据行现已左对齐；排序表头增加半透明底，并与上方菜单栏左右对齐。
- **天气胶囊修复**：移除了重复的标题图标，胶囊模式下只显示天气 emoji。
- **胶囊悬停遮罩**：隐藏胶囊右侧边缘的半透明悬停遮罩（交互行为不变）。

## Test Checklist

- Verify About shows `1.3.2`, then test fresh install and overwrite install on a clean Windows 11 x64 PC.
- Switch language to Japanese, German, and Brazilian Portuguese — verify all UI text is localized.
- Open the search popup, type a query, verify type sort/filter and recommended-apps panel work correctly.
- Delete `%LocalAppData%\DeskBox\data\onboarding.json` and restart — verify the new five-step onboarding appears.
- Open the weather widget, search for a small city — verify the expanded database returns results.
- With QuickLook running, press Space on a file in DeskBox — verify QuickLook preview works and QuickLook remains functional afterward.
- Test tray show/hide (F7) rapidly 10 times — verify no animation tearing or wallpaper loss.
- Drag a dead `.lnk` shortcut into a file widget — verify it is detected and can be cleaned up.
- Test overwrite upgrade from 1.3.1 — existing widgets, settings, and managed storage should survive.
- Run the installer on a non-Chinese locale (e.g. English/Japanese) — verify the language dialog appears, the chosen language is applied to the installer UI, and DeskBox defaults to it on first run.
- On an ARM64 PC (or a VM), run `DeskBox_Setup_1.3.2_arm64.exe` — verify it detects the ARM64 .NET 10 / Windows App Runtime and launches.
