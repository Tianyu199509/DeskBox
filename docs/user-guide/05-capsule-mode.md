# DeskBox 胶囊模式与组合排列指南

胶囊模式让格子在不使用时收起，只保留最有价值的信息。它适合希望天气、音乐、待办和文件状态长期可见，但不想让完整窗口占满桌面的用户。

> [图片占位：同一个桌面的展开状态和胶囊状态对比。标出每个胶囊对应的文件数量、待办、音乐和天气信息]

## 开启胶囊模式

进入 `设置 > 胶囊模式`，打开「允许格子收起」。

总开关默认关闭。关闭时，格子保持展开，下方配置冻结但不会被清空。再次开启后，会恢复之前的内容模式、排列、动画和单格配置。

开启后，可以从格子标题栏的收起按钮切换状态。文件格子标题栏右键菜单也提供展开方式入口。

## 三种展开方式

### 保持展开

格子不自动进入胶囊。适合需要长期操作的主要工作格子，也可以作为单个格子的例外设置。

### 点击展开

点击胶囊进入展开状态，再使用收起按钮返回。它最容易理解，也最不容易误触，推荐新用户默认使用。

### 悬停展开

鼠标停留在标题中间的可展开区域，达到设定延迟后自动展开。移出并满足收起条件后自动返回胶囊。

标题区域被分为三部分

- 左侧图标区，用于识别和拖动，不触发自动展开。
- 中间标题空白区，用于悬停展开。
- 右侧操作区，根据当前格子实际按钮自适应宽度，包含按钮和最右侧六点拖动手柄，不触发自动展开。

操作区平时保持克制，鼠标进入标题后再显示必要提示。这样用户能看懂哪里可以移动，哪里可以展开。

> [图片占位：胶囊标题栏命中区域示意。使用三种半透明颜色标出左侧图标区、中间展开区、右侧操作与六点拖动区]

## 悬停响应

悬停响应控制的是「停多久才展开」和「移出多久才收起」，不等于动画速度。

预设建议

| 预设 | 适合场景 |
| --- | --- |
| 灵敏 | 熟悉胶囊交互，希望快速查看 |
| 平衡 | 日常使用，响应与防误触之间折中 |
| 防误触 | 经常经过胶囊或需要拖动 |
| 自定义 | 分别调整展开和收起延迟 |

如果经常划过就展开，改成防误触。如果鼠标停住却觉得等待太久，改成灵敏或减少自定义展开延迟。

## 收起后显示什么

### 关键信息

每种格子展示对它最有价值的内容。

- 文件，格子名称和项目数量。
- 待办，下一条有价值的任务或状态。
- 随记，摘要或数量。
- 音乐，封面和正在播放的标题。
- 天气，当前天气信息。

### 简要摘要

以格子名称、数量和状态为主，信息更稳定、更统一。

### 仅图标和标题

只保留识别格子所需的内容，适合极简桌面和较窄胶囊。

备注正文不会使用跑马灯。音乐标题过长时沿用音乐格子的单向跑马灯逻辑，避免来回滚动造成疲劳。

## 隐私保护

开启「隐藏具体内容」后，待办和随记胶囊不展示正文。天气、数量和不敏感的状态仍可保留。

隐私设置和内容模式是两个独立选项。你可以让天气继续显示关键信息，同时隐藏待办和随记内容。

## 宽度关系

### 保持同宽

胶囊与展开格子使用相同宽度。调整任一状态的宽度，另一状态同步。

这是默认模式，适合大多数用户。展开和收回时边界关系清楚，鼠标落点也更稳定。

### 独立调整

胶囊和展开状态分别保存宽度。可以让胶囊很短，展开后保持较大的工作区域。

独立模式更灵活，也更需要理解锚点。调整右边框时左边应该保持不动，调整左边框时右边保持不动。胶囊位置不会因为修改展开尺寸而被重新计算。

## 展开锚点

展开和收起都以胶囊所在边角为锚点。

- 靠近屏幕左侧，优先向右展开。
- 靠近屏幕右侧，优先向左展开。
- 靠近顶部，优先向下展开。
- 靠近底部，优先向上展开。

当横向和纵向空间同时受限时，DeskBox 会结合工作区选择可见区域更大的方向。方向可以变化，但胶囊自身的锚点不应漂移。

动态布局格子在展开前会先锁定目标布局，天气不会在动画途中从小布局一路切换到大布局，音乐也不会在中间尺寸反复换视图。

## 独立摆放和组合排列

### 独立摆放

每个胶囊有自己的位置。适合把天气放右上、音乐放底部、待办放左侧等自由布局。

拖动胶囊改变位置时，DeskBox 记录收起状态位置，不修改展开状态尺寸。调整胶囊宽度时会显示左右参考线，帮助与其他格子或屏幕边界对齐。

### 组合排列

多个胶囊组成一条可整体移动的栏。

可以设置

- 胶囊顺序
- 悬浮或贴屏幕边缘
- 顶部、底部、左侧或右侧位置
- 自动、横向或纵向排列
- 胶囊之间的间距

普通用户建议方向保持自动。顶部和底部通常采用横向，左侧和右侧通常采用纵向。

组合栏顺序可以在设置中调整。切回独立摆放时，DeskBox 会恢复之前保存的自由位置，而不是把所有胶囊留在组合栏原地。

## 动画

动画设置控制展开和收回的运动感觉。

| 模式 | 说明 |
| --- | --- |
| 快速 | 响应直接，适合高频操作 |
| 标准 | 默认平衡状态 |
| 舒缓 | 时间更长，视觉更从容 |
| 关闭 | 立即切换状态 |
| 自定义 | 手动调整过渡时长 |

动画速度和悬停响应不要混淆。动画决定开始后运动多久，悬停响应决定何时开始。

如果使用减少动态效果的 Windows 辅助设置，DeskBox 的实际动画表现可能随系统能力调整。

## 拖入时临时展开

把文件、图片、文字或附件拖到胶囊时，格子会临时展开。

流程

1. 内容进入胶囊有效区域。
2. DeskBox 展开真实格子，让你确认目标。
3. 松开后执行添加。
4. 拖放结束，格子恢复原收起状态。

临时展开不会保存为用户主动展开，也不会修改胶囊位置或展开尺寸。

如果只是想移动胶囊，请从图标区、右侧六点手柄或不会触发展开的标题操作区开始拖动。

## 单格设置

单个格子可以覆盖全局的展开方式、宽度和位置。覆盖后，全局设置变化可能不会作用于它。

`设置 > 胶囊模式 > 已单独设置的格子` 会按格子列出覆盖内容，而不是只显示数量。可以逐个恢复跟随全局。

当出现「其他格子都生效，只有这个格子不一样」时，先检查单格覆盖。

## 推荐预设

### 稳定入门

- 点击展开
- 关键信息
- 保持同宽
- 独立摆放
- 标准动画

### 右侧信息栏

- 悬停展开
- 平衡响应
- 关键信息
- 保持同宽
- 组合排列，贴右侧，方向自动

### 极简桌面

- 点击展开
- 仅图标和标题
- 隐藏具体内容
- 独立宽度
- 组合排列，贴顶部或底部
