# DeskBox 文件自动叠放与 QuickLook 指南

自动叠放用于整理文件格子中的视觉结构。它不会移动、重命名或新建真实文件，只把相关项目在界面中收成一组。

当一个格子只有五六个文件时，不需要叠放。文件超过十几个，类型开始混杂时，叠放才会真正有价值。

> [图片占位：同一文件格子开启自动叠放前后对比。右侧显示图片、办公文档、压缩包三个叠放和两个未匹配文件]

## 开启自动叠放

进入 `设置 > 文件格子 > 自动叠放`，打开总开关。

全局设置可以作为默认值。单个文件格子可以在标题栏菜单中选择跟随全局、关闭或使用自己的分组方式。存在单格覆盖时，全局修改不一定作用于该格子。

## 分组方式

### 按文件类型

DeskBox 根据文件类别建立图片、视频、音频、文档、压缩包、快捷方式等叠放。适合普通下载和素材格子。

### 按修改时间

根据文件修改时间分组。适合一个持续工作的目录，用来区分最近修改和较早内容。

### 自定义格式规则

用扩展名建立自己的分类。适合专业文件和项目工作流。

示例

| 叠放名称 | 扩展名 |
| --- | --- |
| 设计源文件 | `.psd .ai .fig .sketch` |
| 办公文档 | `.docx .xlsx .pptx .pdf` |
| 视频工程 | `.prproj .aep .drp` |
| 压缩与交付 | `.zip .7z .rar` |
| 字幕 | `.srt .ass .vtt` |

扩展名可以使用空格、逗号或换行分隔。保存时会统一格式并去除重复项。

## 规则优先级

自定义规则从上到下匹配。一个文件只进入第一条命中的规则。

例如 `.pdf` 同时出现在「办公文档」和「客户交付」，排在上面的规则会获得 PDF。可以使用上下移动操作调整顺序。

规则顺序不是装饰。它直接决定结果。

建议把范围更具体的规则放在上面，范围更广的放在下面。

## 实时预览

每条规则会显示当前格子中的命中数量，并预览匹配内容。预览为零时，检查

- 扩展名是否写错。
- 是否忘记分隔多个扩展名。
- 文件是否已经被更高优先级规则命中。
- 当前设置预览的是哪个文件格子。
- 文件是否为文件夹、快捷方式或没有扩展名的项目。

修改规则名称或扩展名后，预览会同步更新。确认命中结果再离开设置页。

> [动图占位：输入 `.psd .ai .fig`，预览数量从 0 变为实际文件数，随后把规则拖到更高优先级并观察另一规则数量变化]

## 形成数量

可以设置至少 2、3 或 5 个同组文件时才形成叠放，默认 3。

- 2，容易形成叠放，桌面最紧凑。
- 3，默认平衡值。
- 5，只收起数量较多的类别，保留更多散开文件。

形成数量是当前分组的通用门槛，不是每条自定义规则单独设置。

## 未匹配文件

自定义规则无法覆盖所有内容。可以选择

- 保持散开，未匹配文件继续单独显示。
- 收入其他，把未匹配项目统一放进「其他」叠放。

保持散开适合希望发现异常文件的用户。收入其他适合追求紧凑、规则已经覆盖主要内容的格子。

## 叠放内部排序

叠放内部可以沿用格子顺序，或按名称、加入时间、修改时间等方式排列。显示的选项以当前设置页面为准。

如果你依赖手动拖动顺序，使用跟随格子。希望最近文件优先，使用修改时间。

## 展开与收回

点击叠放后，文件在当前格子内部散开展开。原位置保留明确的收回入口，避免展开和收回使用完全相同的视觉导致误解。

快速点击、在多个叠放之间切换或在动画过程中再次操作时，DeskBox 会终止旧状态并进入新的目标状态。不要在文件仍处于拖动或编辑时强制重新分组。

展开期间，文件仍然属于原格子。叠放不是新文件夹，也不是弹出的独立面板。

## 与真实文件的关系

以下操作不会发生

- 不会为了叠放创建真实子文件夹。
- 不会改变文件扩展名。
- 不会移动文件。
- 不会修改资源管理器中的目录结构。

在叠放中删除、重命名或移动具体文件时，仍然会作用于真实文件。

## QuickLook 空格预览

DeskBox 对 QuickLook 使用被动兼容方式。

生效条件

1. Windows 已安装 QuickLook。
2. QuickLook 进程已经运行。
3. 在 DeskBox 文件格子中选中了真实存在的文件或目录。
4. 按下 Space。

DeskBox 会把预览请求发送给 QuickLook，再按 Space 可以切换关闭。它不会

- 自动查找安装目录。
- 自动启动 QuickLook。
- 在设置里增加集成开关。
- 把 QuickLook 变成 DeskBox 的必需依赖。

没有安装或没有运行 QuickLook 时，Space 不应触发 DeskBox 错误。

QuickLook 项目地址

<https://github.com/QL-Win/QuickLook>

## 推荐规则模板

### 设计素材格子

- 设计源文件，`.psd .ai .fig .sketch`
- 图片，`.png .jpg .jpeg .webp .gif .svg`
- 字体，`.ttf .otf .woff .woff2`
- 交付包，`.zip .7z .rar .pdf`

### 视频项目格子

- 工程，`.prproj .aep .drp`
- 视频，`.mp4 .mov .mkv .webm`
- 音频，`.wav .mp3 .aac .flac`
- 字幕，`.srt .ass .vtt`

### 办公下载格子

- 文档，`.doc .docx .pdf .txt .md`
- 表格，`.xls .xlsx .csv`
- 演示，`.ppt .pptx`
- 压缩包，`.zip .7z .rar`

模板只是起点。规则应根据真实文件调整，不建议为了覆盖所有扩展名建立几十条规则。
