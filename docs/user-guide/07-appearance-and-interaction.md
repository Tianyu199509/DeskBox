# DeskBox 外观、设置搜索与交互配置指南

DeskBox 的设置采用接近 Windows 11 的左侧导航和详细页面。普通用户可以只使用入口卡片上的主要选项，高阶参数再进入详情页。

本指南不会把每个滑块逐项罗列，而是说明选项之间的依赖，以及怎样调整不会让界面变得难读或难操作。

## 使用设置搜索

搜索位于设置窗口标题栏，不是每个页面顶部各放一个输入框。

可以搜索

- 页面名称
- 设置卡片标题
- 设置说明中的关键词
- 中英文界面下对应的本地化名称

搜索支持多个关键词。结果按标题匹配、页面路径和说明相关性排序。

点击具体设置项后，DeskBox 会

1. 进入对应页面。
2. 展开必要依赖。
3. 滚动到目标卡片。
4. 提供短暂定位提示和键盘焦点。

例如搜索「附件 路径」，可以直接定位到待办和随记共用的附件保存方式，而不需要先知道它属于哪个设置分区。

## 主题和 Windows 主题色

主题支持跟随系统、浅色和深色。

使用 Windows 主题色时，DeskBox 跟随系统强调色。Windows 如果开启从壁纸自动取色，DeskBox 的按钮状态、强调线和部分图标也会随系统变化。

关闭跟随后，可以选择自定义主题色。

主题色不等于整个窗口背景色。它主要用于强调状态和交互反馈，窗口背景由材质与主题共同决定。

## 窗口材质

可用材质包括

- 云母
- 云母 Alt
- 轻薄亚克力
- 标准亚克力
- 纯色

这些材质尽量使用 Windows 原生窗口能力。不同 Windows 版本、显卡、节能模式、远程桌面和系统透明效果开关，可能影响最终表现。

材质不是一张固定背景图，所以不要期待每台电脑的采样结果完全一致。

### 云母

适合长期停留在桌面的稳定窗口，背景变化克制，文字可读性较好。

### 云母 Alt

层次更明显，适合希望窗口与桌面背景有更强区分的用户。具体差异依赖 Windows 原生合成效果。

### 轻薄亚克力

更轻、更透，适合小型格子和浅内容。背景复杂时要检查文字对比度。

### 标准亚克力

模糊和材质感更明显，适合希望格子与桌面融合但仍保持边界的布局。

### 纯色

完全不透明。选择纯色后，背景透明度会被固定为 100%，透明度滑块不显示。这样纯色不会被旧透明度值干扰。

## 背景透明度和材质浓度

两个滑块作用不同。

- 背景透明度控制允许透明的背景混合程度。
- 材质浓度调整原生材质的色调和层次感。

并非每种材质都支持相同参数。设置页只在当前材质可用时显示或启用对应控件。

调整原则

- 先选择材质。
- 再确认文字与图标是否清楚。
- 最后微调透明度或浓度。

不要为了更透明牺牲内容可读性。桌面壁纸经常变化时，过低的背景不透明度容易在亮色区域失去边界。

## 边框颜色、粗细和圆角

边框颜色

- 中性，使用不抢眼的系统边界色。
- 主题色，使用当前强调色。
- 无，隐藏可见边框。

边框粗细支持细、中、粗。胶囊复用格子的边框参数，不应单独出现更粗的描边。

圆角支持方形、小圆角、大圆角和系统默认等模式。胶囊、天气和音乐内部媒体表面会尽量跟随格子圆角，避免外框方形而内部仍然保留大圆角。

媒体圆角主要控制音乐封面等媒体内容，不是第二套窗口圆角。

## 显示密度

显示密度提供

- 紧凑
- 标准
- 宽松
- 自定义

预设会同时调整文字、行高、项目高度、内边距和间距等相关值。手动改变任意密度参数后，显示密度自动变为自定义。

密度会实时作用于文件列表、待办和随记。个别正在编辑或动画中的内容可能在交互结束后刷新，以避免布局在鼠标操作中跳动。

推荐

- 小屏幕或格子很多，使用紧凑。
- 普通桌面，使用标准。
- 触控或高分辨率大屏，使用宽松。
- 只有明确需求时再使用自定义。

## 标题栏模式

DeskBox 将显示类格子和交互类格子的标题样式分开配置。

可选模式

- 标准，完整标题栏。
- 紧凑，较低标题栏。
- 悬浮或覆盖，标题位于内容表面之上。
- 隐藏，不显示常规标题栏。

显示类格子通常包括文件和天气，交互类格子包括待办、随记和音乐。

收起按钮始终属于胶囊行为，不受「鼠标悬浮操作按钮」总开关影响。标题栏存在时，收起入口固定在最右侧。标题栏悬浮或隐藏时，DeskBox 使用不与内容按钮冲突的独立收起入口。

隐藏标题后，移动、重命名和右键菜单会更难发现。建议只在布局稳定、已经熟悉操作后使用。

## 标题图标和悬停按钮

标题图标可以选择填充、线性、彩色、隐藏或文字标签等模式。

悬停操作按钮可以配置锁定位置、锁定大小、添加、更多和删除等快捷入口。关闭悬停按钮只隐藏这些快捷操作，不会移除胶囊展开和收起能力。

右侧操作区宽度根据实际按钮自适应。没有额外按钮时，只保留最右侧六点拖动手柄的位置，避免挤占中间悬停展开区域。

## 全局快捷键

默认 F7 显示或隐藏全部格子。

在 `设置 > 快捷与交互 > 全局快捷键` 中可以关闭或重新录制。录制时按 Esc 取消。

选择快捷键时避免与以下类型冲突

- 游戏和显卡覆盖层
- 输入法快捷键
- IDE 调试键
- 屏幕录制和截图软件
- 笔记本厂商功能键

快捷键注册失败时，换一个组合并完全退出后重启 DeskBox。

## 格子层级

动态层级适合大多数用户。格子平时靠近桌面，需要操作时再被提升，完成交互后交还 Windows 管理。

桌面固定模式让格子更像桌面背景的一部分，不频繁浮到其他应用上方。

格子展开、拖放、菜单和尺寸调整期间可能临时提升层级，确保操作不被其他格子遮挡。交互结束后恢复对应模式。

如果展开格子被其他格子盖住，或切换应用后层级异常，参考 [常见问题与故障排查](10-troubleshooting.md)。

## 显示和隐藏动画

托盘或 F7 唤起全部格子时，动画应移动真实格子窗口，不是在最终位置先画一层半透明占位。

动画选项决定出现和消失方式。多显示器、远程桌面或系统减少动态效果可能改变表现。

如果追求最快响应，可以关闭动画。动画关闭只影响视觉过渡，不改变位置保存和层级逻辑。

## 新手与高级用户的配置边界

新手只需要调整

- 主题
- 材质
- 显示密度
- 全局快捷键
- 胶囊总开关

高级用户再调整

- 材质浓度和透明度
- 边框颜色与粗细
- 标题栏分类样式
- 悬停操作按钮
- 胶囊延迟、动画和单格覆盖
- 自定义密度

如果界面调整后出现难以判断的状态，优先回到预设，不要同时改变所有高级参数。
