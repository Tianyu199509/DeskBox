# DeskBox 待办格子使用指南

DeskBox 待办适合放在桌面边上的个人轻任务。它不代替团队项目管理，而是帮助你快速记住今天要处理、稍后要回复或需要随文件一起跟进的事情。

> [图片占位：待办格子完整界面。展示今天、重要、已完成标签，列表中包含普通任务、带日期任务和带附件任务]

## 创建待办格子

先在 `设置 > 功能格子` 中确认待办已启用，再通过托盘右键菜单创建待办格子。

待办属于单例功能格子。已经存在时，再次创建会优先定位现有格子，而不是不断生成重复窗口。

## 新建任务

在待办列表使用新增入口，输入内容后保存。

默认键盘行为

- Enter 换行
- Ctrl+Enter 保存

可以在 `设置 > 功能格子 > 待办` 中互换为

- Enter 保存
- Ctrl+Enter 换行

多行任务适合补充简短上下文，但列表主要用于扫读。更长的说明和文件资料建议放到附件或关联随记中。

## 列表和详情

列表显示任务摘要、完成状态、颜色、日期等关键信息。每条任务展示 1 到 10 行，可以在设置中调整，默认是 2 行。

点击任务进入格子内详情页。详情页可以编辑

- 完整正文
- 重要状态
- 截止日期
- 提醒
- 重复规则
- 颜色标记
- 多个附件

完成按钮、固定操作、复制和删除等按钮保持紧凑尺寸，便于在小格子中使用。

> [动图占位：新建任务，Ctrl+Enter 保存，点击进入详情，设置日期和附件，再返回列表]

## 完成与恢复

点击任务左侧完成控件，将任务标记为已完成。已完成任务是否继续显示，可以在待办设置中控制。

如果标签栏显示「已完成」，可以在该视图集中查看和恢复任务。也可以把未完成任务拖到「已完成」标签，或把已完成任务拖到「进行中」标签恢复。

## 截止日期

任务可以设置今天、明天、本周或自定义日期。日期用于筛选和提醒，也是「今天」「本周」「本月」视图的基础。

把任务直接拖到「今天」标签，会把截止日期设为今天。当前版本的标签拖放操作目标包括

- 进行中，恢复为未完成
- 今天，将截止日期设为今天
- 重要，标记为重要
- 已完成，标记为完成

「本周」和「本月」用于查看时间范围，不作为直接拖放目标。

## 提醒

设置截止日期后，可以为任务配置提醒。到达提醒时间时，DeskBox 使用 Windows 通知提醒。

提醒的提前量和默认行为可以在待办设置中调整。系统通知是否能正常展示，还受 Windows 通知权限、专注模式和应用后台状态影响。

如果没有收到提醒，按以下顺序检查

1. 任务是否有有效日期和提醒时间。
2. DeskBox 是否正在运行。
3. Windows 是否允许 DeskBox 发送通知。
4. 专注模式是否隐藏了通知。

## 重复任务

重复规则适合每日检查、每周汇报、每月账单等周期性事项。

完成当前任务后，DeskBox 根据重复规则生成或推进下一次任务，并保留必要的重复历史。修改重复任务前，应确认是在调整当前实例还是整个后续习惯。

对于复杂的跨项目依赖、工作日计算和多人分配，应继续使用专业任务工具。

## 颜色标记

待办支持多种颜色。颜色没有强制语义，可以按自己的规则使用。

推荐选择一种简单体系

- 红色，今天必须处理
- 黄色，等待他人或等待材料
- 蓝色，深度工作
- 绿色，生活与个人事务

不要给八种颜色都规定复杂含义。颜色的作用是帮助扫读，不是建立第二套项目管理系统。

顶部颜色标签既可以筛选，也可以拖到任务卡片上直接赋色。任务已经有颜色时，拖入新颜色会替换原标记。

> [动图占位：拖动蓝色标签到任务卡片，再拖动红色标签替换，最后点击红色筛选]

## 附件

一条任务可以关联多个文件或图片。可以在详情页添加，也可以直接把文件拖到任务卡片。

附件保存有两种模式

- 关联原路径，不复制文件，默认模式。
- 复制到 DeskBox，由 DeskBox 保存一份托管附件。

原文件位置稳定、体积较大时，适合关联原路径。微信、浏览器或临时目录拖出的文件，担心来源很快失效时，适合复制到 DeskBox。

复制带附件的待办时，剪贴板内容会包含任务文本与附件路径，并按当前界面语言添加清楚的字段名称。批量复制会分隔每条任务。

## 标签页显示

可以选择展示以下视图

- 全部
- 进行中
- 今天
- 本周
- 本月
- 重要
- 已完成

也可以隐藏整条标签栏。至少应保留一个有效默认视图，否则设置会回退到可用组合。

推荐配置

- 轻量使用，只显示今天、重要、已完成。
- 周计划，显示进行中、今天、本周、已完成。
- 任务较多，保留全部视图并使用颜色筛选。

## 排序与批量操作

列表支持拖动排序。筛选状态会影响可见项目，调整顺序前应确认当前所在视图。

通过 Ctrl、Shift 或框选选择多个任务后，可以批量执行复制、删除、颜色、日期、提醒和重复等操作。批量修改日期或完成状态前，先检查选择数量。

## 胶囊状态

待办胶囊在关键信息模式下优先展示最有价值的任务信息。标题始终限制为一行，较长内容会省略，避免胶囊高度被正文撑大。

隐私模式可以隐藏任务正文，只保留图标、名称、数量或状态。拖入文本或文件时，胶囊会临时展开并显示「添加到待办」语义。

## 推荐工作流

每天开始工作时，只做三件事

1. 把脑子里怕忘的事情快速写下来。
2. 把真正今天要完成的任务拖到「今天」。
3. 给必须处理的一到三件事标记重要或颜色。

下班前完成或重新安排未处理任务。DeskBox 待办适合保持轻量，长期项目资料仍然应该回到项目管理工具或日历。
