# DeskBox 全网 SEO 优化与渠道推广方案

## 一、SEO 优化方案

### 1.1 官网 SEO（基础）

**技术 SEO**
- Next.js SSR/SSG 确保所有页面可被爬虫索引
- 自动生成 `sitemap.xml` 和 `robots.txt`
- 结构化数据：`SoftwareApplication` schema（应用名、版本、下载链接）
- Open Graph / Twitter Card 元标签
- 规范 URL 结构：`/features`、`/download`、`/changelog`
- 页面加载速度优化（Core Web Vitals）

**内容 SEO**
- 每个功能页独立 URL + 中英文描述
- 更新日志自动从 CHANGELOG.md 生成
- 长尾关键词覆盖：
  - 中文：桌面整理工具、Windows 11 桌面格子、文件管理小工具、桌面文件收纳、剪贴板管理器
  - English: Windows 11 desktop organizer, desktop widget, file manager, clipboard manager, desktop file organizer

### 1.2 GitHub SEO

- README 中英文完善（已完成）
- Topics 标签：`desktop-organizer`、`windows-11`、`winui-3`、`file-manager`、`clipboard-manager`
- 项目描述优化
- 定期更新 CHANGELOG
- 鼓励用户 Star 和 Fork

### 1.3 应用商店 SEO（如有上架计划）

- Microsoft Store：标题、描述、关键词、截图
- 各软件下载站：华军、太平洋、天空下载站等

## 四、执行顺序

1. **官网部署** — 绑定域名，提交搜索引擎
2. **微信公众号** — 持续发布产品文章
3. **知乎** — 回答桌面整理相关问题
4. **少数派** — 发布产品体验文章
5. **Product Hunt** — 英文市场发布
6. **B站** — 产品演示视频
7. **其他渠道** — 根据效果逐步扩展
